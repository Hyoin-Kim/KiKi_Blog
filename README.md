
<h2>지브리 테마 기반의 블로그 기능의 웹페이지</h2>

<p>프론트 단에서 jQuery를 활용한 자바스크립트, 백 앤드 단에서 laravel 프레임워크를 활용한 PHP 로 개발</p>
<p>MVC패턴을 이용하여 CRUD 활용</p>
<p>메모(Board), 앨범, 친구 추가기능 및 쪽지 소통기능, 게시물 좋아요/ 싫어요 및 댓글 기능, 알람 기능 구축</p>
